# Trust & Safety Reports

This repository is for educational and defensive purposes only.

All information contained here is derived from publicly available sources, voluntary victim disclosures (with consent), and platform metadata (timestamps, public profiles).

No private information is published. No coordinates. No residential addresses. Location data is limited to timezone analysis and self‑disclosed affiliation (university).

This content is intended to help community members recognize recurring fraud patterns, protect themselves from similar schemes, and make informed decisions when engaging with unknown parties online. The goal is prevention, not retaliation.

If you believe any content violates platform policies or contains inaccuracies, please open a GitHub issue for review. Requests for removal or correction will be addressed promptly.
